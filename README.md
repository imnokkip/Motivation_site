# 🌍 Мотивационные цитаты, если нужна поддержка

**Каждая цитата находит свой пейзаж. Минималистичный сайт с «умным» фоном**

[![Python](https://img.shields.io/badge/python-3.8%2B-blue)](https://python.org)
[![Flask](https://img.shields.io/badge/Flask-2.0-green)](https://flask.palletsprojects.com/)

---

## 📌 О проекте

Это веб-приложение, которое показывает вдохновляющие цитаты и автоматически подбирает к ним фоновое изображение.

Ты заходишь на сайт — видишь цитату и красивый фон. Обновляешь страницу — цитата и картинка меняются.  

Проект написан на **Flask**, использует **API Ninjas** для цитат и **Wallhaven API** для фоновых изображений.

---

## 🚀 Возможности

- Случайная мотивирующая цитата при каждом обновлении
- Фоновое изображение
- Красивый, адаптивный интерфейс с плавным затемнением
- Минималистичный дизайн

---

## 🖥️ Запуск

### 1. Клонируй репозиторий

```bash
git clone https://github.com/imnokkip/Motivation_site
pip install -r requirements.txt
```

### 2. Получаем api на сайте [api-ninjas](https://api-ninjas.com/api/quotes) и вставляем в app.py вместо ACCESS_KEY

### 3. Запускаем app.py и переходим по адресу http://127.0.0.1:5000
